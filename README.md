# SWproject_AICalibrationDiagnostic
생성형 AI 답변과 사용자 교정본의 변화를 정량적으로 측정하여, AI가 작성한 글을 사용자가 무비판적으로 수용하고 있는지 점검하는 페이지

## 프로젝트 개요
**개발 배경**: 대학 생활과 일상에서 AI가 대신 써주는 글에 지나치게 의존하는 현상을 방지하고자 기획된 생성형 AI 답변 자가 교정 및 의존도 진단 도구이다.
**핵심 기능**: 외부 AI의 원본 답변과 사용자가 수정한 교정본을 공백 기준으로 쪼개어 추가/삭제된 단어를 추적하고, 문장 수(마침표 개수) 변화를 계산하여 'AI 의존도'를 정량적으로 평가한다.

## 사용 방법
1. 첫 번째 입력창에 **점검**을 입력하면 미리 정의된 6가지 시사 논점 중 하나가 랜덤으로 추출된다. 
2. 제시된 논점을 외부 AI(ChatGPT, Claude 등)에 입력하여 답변을 얻어낸 뒤 **'AI 원본 답변'** 란에 복사·붙여넣기한다. 이때 본인이 가장 자주 사용하는 AI를 이용하는 것을 권장한다. 
3. 사용자는 자신의 방식대로 해당 글을 다듬어 우측 **'사용자 주체적 교정본'** 란에 작성한다. 
4. **'진단 실행'** 버튼을 누르면 비교 분석 과정과 AI 의존도에 대한 판단 결과가 즉시 출력된다.
